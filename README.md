# HanHua (喊话助手)

WoW Classic / 时光服 喊话辅助插件。快速组队喊话，支持自定义频道、多段输入、快捷模版。

## 功能

### 发送

- 点击 **发送** 按钮、输入 `/fasong` 或设置快捷键，向选定频道发送喊话
- 支持大喊（YELL）和自定义频道（CHANNEL）
- 按钮 20 秒冷却倒计时，冷却结束后可勾选 **自动** 循环发送
- 发送内容一键 **复制** 到剪贴板

### 输入

五个输入段，按顺序拼接为最终喊话（中间用 `-` 连接），顶部实时预览：

| 段 | 说明 |
|---|---|
| 重要 | 开团关键信息（宝库/P3/P2/P1 包团） |
| 活动 | 副本活动描述（NAXX双龙百元团等） |
| 补充 | RTN分配规则、DPS要求等 |
| 职业 | 招募职业，拼接格式：`来 JLM/NQ/FQ` |
| 备注 | 位置已满、价格、特殊说明等 |

每段独立最大长度限制，实时显示字符数。

### 快捷输入（底部面板）

Tab 切换五个分类，可滚动查看：

- **重要** — 分组列表（宝库/P3/P2/P1），点击填入输入框；支持保存/删除自定义模版
- **活动** — 按阶段（P1-P4）分组的副本活动列表，支持模版保存
- **补充** — 按分组（通用 / DPS / 特殊）的多选 Checkbox，勾选后拼接填入
- **职业** — 按职业分类（T / N / 近战DPS / 远程DPS / 泛称）多选，勾选后格式 `来 FQ/DKT`
- **备注** — 按分组（位置已满 / 价格 / 通用）多选，勾选后拼接

### 频道管理

- 自动读取游戏中加入的自定义频道
- 排除 MeetingHorn 和 BiaoGeYY
- 在界面上以 CheckButton 展示，勾选控制发送目标
- 支持全选/取消全选

### 历史记录

- 自动保存最近 15 条发送记录
- 左键点击还原各段内容到输入框
- 右键删除单条

### 集结号（MeetingHorn）集成

如果安装了 MeetingHorn，发送喊话时自动同步活动说明到集结号的创建面板，并自动发布活动。

### 界面

- 可拖动（按住 Shift 拖 ★ 按钮）
- 左键 ★ 按钮折叠/展开面板
- Alt + 悬停 ★ 显示更新记录
- 窗口位置自动保存

## 配置

编辑 `HanHua_Config.lua` 可自定义各段字符上限和窗口尺寸，修改后 `/reload` 生效：

```lua
HH_config.prefix_max  = 100   -- 活动最大字符数
HH_config.middle_max  = 150   -- 职业最大字符数
HH_config.suffix_max  = 100   -- 备注最大字符数
HH_config.important_max = 40    -- 重要最大字符数
HH_config.tuanbu_max  = 40    -- 补充最大字符数
HH_config.frame_width = 325   -- 面板宽度
HH_config.frame_height= 310   -- 面板高度
HH_config.qi_height   = 400   -- 快捷面板高度
```

快捷输入项配置在 `HanHua_QuickInput.lua`，JSON 风格 Lua 表，直接增删改分类和选项即可。

## 命令

- `/fasong` — 发送喊话
- 游戏按键设置 → HanHua喊话助手 → 发送喊话 绑定快捷键

## 兼容性

支持所有 WoW 版本（Classic / 时光服 / Retail），TOC 包含 11508–120000 接口版本号。
